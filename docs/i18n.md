## Uso

### 1. Usar traducciones en un componente

```tsx
import { useTranslations } from "@/hooks/useI18n";

function MiComponente() {
  const t = useTranslations();

  return (
    <div>
      <h1>{t.hero.subtitle}</h1>
      <p>{t.common.tagline}</p>
    </div>
  );
}
```

### 2. Cambiar idioma

```tsx
import { useI18n } from "@/hooks/useI18n";

function MiComponente() {
  const { language, setLanguage } = useI18n();

  return (
    <button onClick={() => setLanguage(language === "es" ? "en" : "es")}>
      Cambiar idioma
    </button>
  );
}
```

### 3. Agregar un nuevo idioma

1. Crear archivo de traducciones (ej: `src/lib/i18n/fr.ts`)
2. Actualizar el tipo `Language` en `types.ts`
3. Agregar las traducciones al objeto en `index.ts`

### 4. Agregar nuevas traducciones

1. Actualizar la interfaz `Translations` en `types.ts`
2. Agregar las traducciones en todos los archivos de idioma
